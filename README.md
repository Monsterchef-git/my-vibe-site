# John Herrera | Vibe-Coder & Chef

Portfolio landing built with Next.js for a personal brand that combines culinary direction and digital product craft.

## Overview

This site presents two sides of the same profile:

- Creative culinary services and private dining experiences in Medellin
- Conversion-focused digital work, landing pages, and product direction

The visual system leans into a dark editorial look with lime accents, grain overlays, blurred glass panels, and terminal-inspired UI details.

## Stack

- Next.js 16.2.1
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production Check

```bash
npm run lint
npm run build
```

## Project Structure

- `src/app`: App Router entry points, global layout, and page composition
- `src/components`: reusable primitives and portfolio sections
- `public/images`: visual assets used across featured projects and culinary sections

## Notes

- This repo is a portfolio/marketing site, not a reusable package
- Environment variables are not required for the current build
